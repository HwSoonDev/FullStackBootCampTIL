
Tailwind 폰트 굵기 클래스 종류

|-------------------|----------------------|----------------|
| font-thin         | 100                  | 매우 얇음     |
| font-extralight   | 200                  | 아주 가벼움   |
| font-light        | 300                  | 가벼움       |
| font-normal       | 400                  | 기본(normal)  |
| font-medium       | 500                  | 중간 굵기    |
| font-semibold     | 600                  | 반굵게       |
| font-bold         | 700                  | 굵게         |
| font-extrabold    | 800                  | 아주 굵게    |
| font-black        | 900                  | 가장 굵게    |
