| Next.js 기반 세팅 종류 | 설명                                                                                                                                                                                                                                                                       |
| ---------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| TypeScript       | 자바스크립트 타입 검사                                                                                                                                                                                                                                                             |
| Eslint           | 문법 오류 감지, 코드 스타일 통일<br>(일반적으로 prettier와 함께 사용 시 필요)                                                                                                                                                                                                                      |
| ReactComplier    | 리액트 기반                                                                                                                                                                                                                                                                   |
| Tailwond         | 컴포넌트 태그에 직접 작성하는 CSS 스타일                                                                                                                                                                                                                                                 |
| src/ directory   | 소스 코드의 루트 폴더 설정 <br>(src: 빌드 시 위치가 변하는 상대 경로)<br>(public: 이미지 등 절대 경로)                                                                                                                                                                                                   |
| App Router       | 웹페이지 폴더 기반 라우팅                                                                                                                                                                                                                                                           |
| import alias     | 상대 경로(`../../../components/Button`) 대신 **별칭 경로**(`@components/Button`)를 쓰도록 해주는 역할<br><br>{<br>  "compilerOptions": {<br>    "baseUrl": "./src",<br>    "paths": {<br>      "@components/*": ["components/*"],<br>      "@utils/*": ["utils/*"]<br>    }<br>  }<br>}<br> |
