```js
// src/app/providers.jsx
"use client";

import ThemeProvider from "@/context/ThemeProvider";

export default function Providers({ children }) {
  return (
    <ThemeProvider>
      {children}
    </ThemeProvider>
  );
}
```

- 클라이언트 컴포넌트로 Provider를 만들어 전체 앱에 컨텍스트를 제공한다.
- 최상위 레이아웃(RootLayout)에서 이 Provider를 import하여 children을 감싸는 방식으로 모든 페이지와 컴포넌트에 컨텍스트를 제공한다.
- **=> 클라이언트 컴포넌트로 provider를 만들어서 최상위(혹은 컨택스트가 필요한 상위) 레이아웃에 적용하여 사용한다.**
