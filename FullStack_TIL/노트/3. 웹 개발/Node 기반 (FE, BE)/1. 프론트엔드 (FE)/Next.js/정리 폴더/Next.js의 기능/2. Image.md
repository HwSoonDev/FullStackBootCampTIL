## Image 컴포넌트

`next/image`는 이미지를 최적화 해주는 고급 컴포넌트.

내부적으로는 이미지 요청을 Next.js 서버로 보내고, 그 서버가 
- 이미지 리사이징, 
- 포맷 변환(WebP 등), 
- 캐싱 
등을 한 뒤에 최종 이미지를 응답한다.

이미지 최적화 때문에 사전의 너비, 높이 설정이 필요하고, 이미지는 `public/image` 폴더에 보관하여, `const imageURL`로 정적 정의하여 사용하는 것이 일반적이다.

이미지 최적화 기능 때문에 Next.js의 Image 컴포넌트를 쓰는게 정말 중요하다.
### 파라미터

`src` : 이미지 소스 URL
`alt` : 이미지의 이름 (이미지가 안 불러 와질 때 보여줄 텍스트)
`width` = 너비
`height` = 높이

위 4가지는 필수 

추가)
`placeholder`: 이미지가 불러와지는 동안 보여줄 기본 이미지 설정
`blurDataURL`: 블러 이미지용 이미지

'blur'를 넣으면 블러 이미지를 보여주고
정적 이미지일 경우 해당 이미지의 블러 상태.
동적 이미지일 경우 `blurDataURL`도 설정해주어야 한다.

```js
// 서버 컴포넌트나 getStaticProps 같은 서버 사이드 환경에서 사용
import { getPlaiceholder } from 'plaiceholder';
import Image from 'next/image';

// 서버 컴포넌트에서 사용하는 예시
async function MyImageComponent() {
  // 외부 이미지 URL
  const imageUrl = 'https://example.com/profile.jpg';
  
  // 이미지 데이터 가져오기
  const response = await fetch(imageUrl);
  const buffer = await response.arrayBuffer();
  
  // Plaiceholder를 사용하여 blur 데이터 생성
  const { base64 } = await getPlaiceholder(Buffer.from(buffer));
  
  return (
    <div>
      <Image
        src={imageUrl}
        alt="프로필 이미지"
        width={500}
        height={300}
        placeholder="blur"
        blurDataURL={base64}
      />
    </div>
  );
}

export default MyImageComponent;
```
