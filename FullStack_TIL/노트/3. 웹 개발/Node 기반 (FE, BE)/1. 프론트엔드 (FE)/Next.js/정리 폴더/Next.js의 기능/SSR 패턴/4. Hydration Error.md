**서버에서 렌더링된 HTML과 클라이언트에서 렌더링을 시도할 때의 결과물이 일치하지 않을 때** 발생하는 오류

→ 서버와 클라이언트 사이에 렌더링 불일치

### 예시 1: 클라이언트 전용 API 사용

```jsx
'use client'; // 클라이언트 컴포넌트

function UserLocation() {
  // 서버에서는 window 객체에 접근할 수 없음
  const [location, setLocation] = useState(window.location.href); // 🚨 Hydration Error
  
  return <div>현재 URL: {location}</div>;
}
```

해결

```jsx
'use client';
import { useState, useEffect } from 'react';

function UserLocation() {
  const [location, setLocation] = useState('');
  
  useEffect(() => {
    // 클라이언트에서만 실행
    setLocation(window.location.href);
  }, []);
  
  return <div>현재 URL: {location}</div>;
}
```

### 예시 2: 서버와 클라이언트의 초기 상태 불일치

```jsx
'use client';

function RandomNumber() {
  // 서버와 클라이언트에서 각각 다른 값 생성
  const [number] = useState(Math.random()); // 🚨 Hydration Error
  
  return <div>랜덤 숫자: {number}</div>;
}
```

해결
```jsx
'use client';

function RandomNumber() {
  // 서버와 클라이언트에서 각각 다른 값 생성
  const [number, setNumber] = useState(0);
  
  useEffect(()=>{
	  setNumber(Math.random());
  }, [])
  
  return <div>랜덤 숫자: {number}</div>;
}
```

### 예시 3: 조건부 렌더링 불일치

```jsx
'use client';

function ConditionalComponent({ user }) {
  // 서버에서는 false, 클라이언트에서는 true가 될 수 있음
  const isLoggedIn = typeof window !== 'undefined' ? 
    localStorage.getItem('token') !== null : false; // 🚨 Hydration Error
  
  return (
    <div>
      {isLoggedIn ? <p>환영합니다, {user}님!</p> : <p>로그인이 필요합니다</p>}
    </div>
  );
}
```

위 컴포넌트에서는 로그인한 유저인지 판단하는 로직을 사용하고 있지만,
로그인 유저더라도 서버에서는 false가 나올 수 있다.

가장 대중적인 방법으로는 dynamic import 로 서버 환경에서는 아예 실행되지 않게 하는 방법이 있다.
```jsx
'use client';

import { useState, useEffect } from 'react';
import dynamic from 'next/dynamic';
// import ClientOnlyComponent from "./ClientComponent"
// 클라이언트에서만 렌더링되는 컴포넌트
const ClientOnlyComponent = dynamic(() => import('./ClientComponent'), {
  ssr: false // 서버 사이드 렌더링 비활성화
});

function SafeComponent() {
  const [isClient, setIsClient] = useState(false);
  
  useEffect(() => {
    setIsClient(true);
  }, []);
  
  return (
    <div>
      {isClient ? <ClientOnlyComponent /> : <p>로딩 중...</p>}
    </div>
  );
}
```
