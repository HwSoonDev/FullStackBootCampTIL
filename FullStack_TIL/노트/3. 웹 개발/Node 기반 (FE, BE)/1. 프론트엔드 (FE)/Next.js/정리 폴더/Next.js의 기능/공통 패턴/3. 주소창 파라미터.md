## 서버 컴포넌트에서

### Params

```jsx
export default function Page({ params }){
	const { id } = params
}
```

컴포넌트 인자로 params를 설정해주면, 따로 넘겨주지 않아도 주소창의 파라미터를 받아온다.
## 클라이언트 컴포넌트에서
### useParams

`[id]` 같은 경로 파라미터를 사용할 수 있다.
```jsx
"use client"

import { useParams } from 'next/navigation'

export default function Page({}){
	const { id } = useParams();
}
```

### useSearchParams

쿼리 파라미터를 읽어오는 훅이다.

예시에서,
`http://localhost:3000?filter="` 
이런 주소라면 filter라는 쿼리를 받아올 수 있다.

```jsx
'use client';

import { useParams, useSearchParams } from 'next/navigation';

export default function Page() {
  const params = useParams(); // path param
  const searchParams = useSearchParams(); // query param

  const id = params.id;
  const filter = searchParams.get('filter');

  return (
    <div>
      <p>게시글 ID: {id}</p>
      <p>필터: {filter}</p>
    </div>
  );
}

```
