| 상태 코드 |                           |
| --------- | ------------------------- |
| 100번대   | 정보성 상태 코드          |
| 200번대   | 성공 상태 코드            |
| 300번대   | 리디렉션 상태 코드        |
| 400번대   | 클라이언트 에러 상태 코드 |
| 500번대   | 서버 에러 상태 코드       |

| 200번대 | 이유 구문  | 설명                                                   |
| ------- | ---------- | ------------------------------------------------------ |
| 200     | OK         | 요청이 성공했음                                        |
| 201     | Created    | 새로운 자원이 생성됨                                   |
| 202     | Accepted   | 요청을 잘 받았으나, 아직 요청한 작업을 완료하지 않았음 |
| 203     | No Content | 요청에 성공했지만, 메세지 본문으로 표시할 내용이 없음  |

| 300번데 | 이유 구문             | 설명                                             |
| ------- | --------------------- | ------------------------------------------------ |
| 301     | Moved Permanently     | 영구적 리디렉션 - 재요청 메서드가 변경될 수 있음 |
| 308     | Permanent redirection | 영구적 리디렉션 - 재요청 메서드가 변경되지 않음  |
| 302     | Found                 | 일시적 리디렉션 - 재요청 메서드가 변경될 수 있음 |
| 303     | See Other             | 일시적 리디렉션 - 제요청 메서드가 GET으로 변경됨 |
| 307     | Temporary Redirect    | 일시적 리디렉션 - 재요청 메서드가 변경되지 않음  |
| 304     | Not Modified          | 캐시 - 자원이 변경되지 않음                      |

영구적 리디렉션은 자원의 위치가 영구적으로 바뀌었기 때문에 기존의 URL을 기억할 필요가 없고, 일시적 리디렉션은 기존의 URL을 기억해둘 필요가 있다.
308은 301처럼 요청 메서드가 바뀔 가능성이 있는 애매함을 보완하기 위해 나온 코드다.
304는 별다른 처리를 안해도 케트워크 창에 뜨는 걸 본적이 있다. 자원이 바뀌지 않으면 자동으로 보내는 듯 하다.

| 400번대 | 이유 구문          | 설명                                            |
| ------- | ------------------ | ----------------------------------------------- |
| 400     | Bad Request        | 요청한 메세지의 내용이나 형식에 문제가 있음     |
| 401     | Unathorized        | 요청한 자원에 대한 유효한 인증이 없음           |
| 403     | Forbidden          | 요청이 서버에 의해 거부됨 ex)충분한 권한이 없음 |
| 404     | Not Found          | 요청한 자원을 찾을 수 없음                      |
| 405     | Method Not Allowed | 요청한 메서드를 지원하지 않음                   |

| 500 번대 | 이유 구문             | 설명                  |
| -------- | -------------------- | --------------------- |
| 500      | Internal Server Error | 요청을 처리할 수 없음 |
| 502      | Bad Gateway           | 중간 서버의 통신 오류 |
