# `GROUP BY` (그룹 집계)
그룹 별로 통계를 구할 떄 사용

![[Pasted image 20260123133612.png]]

```
SELECT 열 이름, 집계함수(열이름)
FROM 테이블
WHERE 조건문
GROUP BY 열이름
```

1) WHERE 절 다음에 쓴다. (ORDER BY 절 이전에 쓴다)
2) GROUP BY에 적은 열 이름 중 적어도 하나는 SELECT에 써야 한다.
   (그룹핑한 열 중 하나는 출력으로 선택해야 한다.)
   
   ![[Pasted image 20260123133917.png]]
1) 그룹핑한 열을 기준으로 집계한 결과를 보여준다.
   
   
예시) 국가 별 고객 수
```sql
SELECT Country, COUNT(CustomerID) as CNT
FROM Customers
GROUP BY Country;
```
