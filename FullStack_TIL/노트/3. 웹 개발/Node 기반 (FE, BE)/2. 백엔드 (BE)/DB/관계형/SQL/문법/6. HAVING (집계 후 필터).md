WHERE가 집계 이전의 전체 데이터에 대한 조건문이었다면,
HAVING은 집계 후 완성된 데이터에 대해 다시 한번 조건문을 거는 용도이다.

```sql
SELECT 컬럼1, 집계함수1(), 집계함수2()
FROM 테이블
WHERE 조건문 (집계 전)
GROUP BY 컬럼1
HAVING 조건문 (집계 후, SELECT로 정의된 컴럼 표에 대하여)
```

당연하게도 집계를 해서 데이터 표를 만든 이후에 HAVING이 들어가야 하기 때문에,
GRUOP BY 뒤에 들어가야 한다.

예시) 국가 별 고객 수 (고객 수 5명 초과인 국가만)
```sql
SELECT Country, COUNT(CustomerID) as CNT
FROM Customers
GROUP BY Country
HAVING COUNT(CustomerID) > 5;
```
