# `COUNT`

```sql
SELECT COUNT(*)
FROM 테이블
WHERE 조건문
```

1) `집계 함수` 중 하나이다.
2) 행(row)의 수를 셀 떄 사용한다. (해당하는 튜플의 수)
3) `COUNT(*)` 또는 `COUNT(열이름)`으로 사용한다.
4) 뒤에 as를 붙이고 이 열의 새로운 이름을 적는다. (as는 문법 상 생략이 가능하다)
5) COUNT(DISTINCT 열이름)을 사용하면 unique한 행의 수를 센다. (중복을 제거,  Set같은 느낌)

(예시)
```sql 
SELECT COUNT(customer_id) as total_customers,
		COUNT(DISTINCT customer_id) as total_unique_customers
FROM Customers
```

집계함수에서 `*`을 사용하면 행 전체를 계산한다. 
위 예시처럼 빈 값이 존재하는 컬럼(Height)이 있어도 컬럼을 따로 지정하지 않으면 행 전체를 보기 때문에 카운트가 4가 아닌 5가 되어 버린다.
빈 값이 있을 수 있는 걸 고려 할 때는 컬럼을 특정 해주자.
SUM, AVG는 컬럼을 지정 하지 않으면 계산이 처음부터 안된다.

---

```sql
SELECT SUM(컬럼), AVG(컬럼), MIN(컬럼), MAX(컬럼)
FROM 테이블
```
# `SUM` : 합계 (빈 행 제외)
# `AVG` : 평균 (빈 행 제외)
# `MIN` : 최솟값
# `MAX` : 최댓값
