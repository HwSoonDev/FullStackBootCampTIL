JOIN이 두 테이블의 컬럼을 그대로 들고 와서 붙이는 거라면,
UNION은 두 테이블의 컬럼들을 합쳐서 하나의 테이블로 만드는 것이다.

```sql
SELECT 열이름
FROM 테이블
UNION
SELECT 열이름
FROM 테이블
```


```sql
SELECT 고객번호, 세일, 품목
FROM 온라인주문
UNION
SELECT 고객번호, 세일, 품목
FROM 가게주문
```

1) UNION 앞의 쿼리 결과와, 뒤의 쿼리 결과를 합쳐준다. (엑셀, 파이썬의 concat)
2) 이어주는 테이블의 열의 갯수가 같고, 열의 순서도 같아야 한다.
3) 열의 이름은 같을 필요는 없지만, 그 열의 데이터 타입은 같아야 한다. (boolean, date, etc)
   (보통 같은 종류의 데이터를 한 컬럼으로 만드므로 의식할 일은 없지만)
4) 완성되는 테이블의 열의 이름은 UNION 앞에 오는 쿼리의 테이블의 열 이름을 따른다.
5) 중복되는 항목이 있을 시에는 자동으로 하나만 출력한다. 중복되는 항목을 모두 출력하고 싶은 경우에는 `UNION ALL`을 사용한다.
