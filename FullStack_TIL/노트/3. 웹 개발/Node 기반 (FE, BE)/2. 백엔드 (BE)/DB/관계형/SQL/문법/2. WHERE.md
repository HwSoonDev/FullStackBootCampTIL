# `WHERE` (필터링)

```sql
SELECT 출력하고 싶은 결과
FROM 테이블
WHERE 조건문
```

예시) 고객 명단의 고객 중에 '멕시코'에 사는 고객 만 가져올 때
WHERE문을 사용할 수 있다. 말 그대로 조건을 걸어서 필터링하는 것이다.

1) 여러 조건 동시 만족 (AND)
2) 여러 조건 중 하나만 만족 (OR)
```sql
WHERE 나라=`멕시코` AND 나이=30 //둘 다 만족하는 고객
WHERE 나라=`멕시코` OR 나이=30 //둘 중 하나라도 만족하는 고객
```
3) 제외하고 싶을 때 (!= 또는 <>)
```sql
WHERE 나라 != `멕시코`
WHERE 나라 <> `멕시코`
//위 두가지 모두 멕시코 사람을 검색에서 제외하는 효과가 있다.
```

4) 부등호 (숫자 비교)
```sql
WHERE 나이 >= 30 AND 나이 < 40
```
수학 기호와 동일하게 동작한다.

5) 하나의 컬럼(열)에서의 다중 조건은 in, NOT in 리스트로 해결 가능

```sql
WHERE 나라='멕시코' OR 나라='미국' OR 나라='일본' OR...
```

이렇게 나타낼 수도 있겠지만, 
같은 컬럼(열)을 공유한다면 

```sql
WHERE 나라 in ('멕시코', '미국', '일본')
```

이렇게 리스크로 간단하게 가능하다.
NOT in는 리스트에 속하지 않는 지 (제외할 때)

```sql
WHERE 나라 NOT in ('멕시코', '미국', '일본') //제외
```
 
6) 문자는 ' '(따옴표) 숫자(정수)는 그냥!
   
   문자와 숫자는 SQL에서 따옴표로 구분한다.

## + `BETWEEN` (범위)

범위를 나타내고 싶을 떄, `나이 >=30 AND 나이 < 40` 이렇게도 가능하지만,
BETWEEN 문법을 사용하면 더 직관적으로 가능하다.

( 30대에 속하는 사람 )
```sql
WHERE 나이 BETWEEN 30 AND 39 
```
특히 `30대인 사람을 제외한다`처럼 범위 내의 결과를 제외 할 때 더 직관적이다.

(30대에 속하는 사람을 제외)
``` sql
WHERE 나이 NOT BETWEEN 30 AND 39 
```

# + `LIKE` (일정 패턴)

일벙 패턴을 찾을 때는 LIKE, NOT LIKE 연산자가 유용하다.

```sql
WHERE 이름 LIKE 'Tom%' //이름이 'Tom'으로 시작하는 사람
WHERE 이름 NOT LIKE '%y' //이름이 'y'로 끝나지 않는 사람
WHERE 이름 LIKE '%or%' //이름 중간 어디든 'or'이 들어가는 사람(처음이나 끝도 상관 없음)
```

참고로 `%`는 복수의 문자를 상징하고, `_`는 하나의 문자를 상징한다.
`'B_c%'` 과 같이 쓰면 B땡c로 시작하는 이름을 검색할 것이다.
