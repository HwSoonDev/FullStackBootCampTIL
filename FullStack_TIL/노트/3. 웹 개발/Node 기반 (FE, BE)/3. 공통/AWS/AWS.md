[[클라우드 서비스]]
[[아마존이 aws를 만든 이유]]
**Amazon Web Services** - 아마존이 제공하는 클라우드 컴퓨팅 서비스
- **설립**: 2006년 (클라우드 컴퓨팅 시장의 개척자)
- **점유율**: 전 세계 클라우드 시장 1위 (약 30%)
- **서비스 수**: 200개 이상의 다양한 서비스


계정 관리
[[ROOT vs IAM (권한 축소)]]

제공 서비스
1. [[EC2]] - 서버 컴퓨터 제공
2. [[S3]] - 파일 업로드/다운로드 DB
3. [[RDS]] - 관계형 데이터베이스 서비스
