## 클라우드 컴퓨팅이란?

### 🤔 클라우드란?

: 쉽게 사용하고 반납할 수 있는 컴퓨터 자원(서버, 저장소, DB 등)

### 🤔 클라우드 컴퓨팅이란?

: `클라우드`에 어디서나 편리하게 접근할 수 있게 해주는 기술

</aside>

### ✨ 클라우드의 5가지 핵심 특징

1. **온디맨드(on-demand) 셀프 서비스** - 필요할 때 바로 사용 가능
2. **광범위한 네트워크 접근** - 어디서든 인터넷으로 접속
3. **리소스 풀링** - 여러 사용자가 자원을 공유하여 효율성 증대
4. **신속한 탄력성** - 사용량에 따라 자동으로 확장/축소
5. **측정된 서비스** - 사용한 만큼만 요금 지불

## 1.2 온프레미스 vs 클라우드

### 🏢 온프레미스 방식 (On-Premise)

: 기업이 자체적으로 서버를 구축하고 운영하는 방식

<aside> ☝

On-Premise를 직역하면, ‘건물(구내) 안에’ 라는 뜻입니다 🙂

</aside>

![image.png](attachment:a4906820-6447-41b5-84a4-ab0df86087e9:image.png)

- **초기 비용**: 서버, 네트워크 장비 등을 직접 구매 (수천만원~수억원)
- **관리**: IT 담당자가 직접 서버실 관리, 하드웨어 교체 (정기적인 하드웨어 교체 및 업그레이드 필요)
- **확장**: 새 서버 주문 → 설치 → 설정 (몇 주~몇 달 소요)
- **위험**: 하드웨어 고장 시 서비스 중단 (백업, 이중화 등 비상 계획 수립과 관리의 복잡성)

### ☁️ 클라우드 방식

- **초기 비용**: 거의 없음 (사용한 만큼만 지불)
- **장기적 비용**: 대여비용을 누적하면 장기적으로는 더 큰 비용이 들 수 있음
- **관리**: 클라우드 업체가 알아서 관리
- **확장**: 클릭 몇 번으로 즉시 확장 (몇 분 내)
- **위험**: 클라우드 업체가 안정성 보장

### 💰 비용 비교 예시

**스타트업이 웹서비스를 런칭한다면?**

|**구분**|**온프레미스**|**클라우드**|
|---|---|---|
|초기 투자비용|서버 구매: 5000만원|0원|
|월 운영비|전기세, 인터넷, 관리비: 200만원|사용량에 따라: 10~200만원|
|확장성|새 서버 구매 필요|클릭 한번으로 확장|

## 1.3 클라우드 컴퓨팅의 장점

<aside> ☝

무조건 장점만 있다고 보기는 어렵지만, 그래도 웬만한 서비스 범위(규모)를 지원하는 서비스라면 거의 모든 니즈를 만족시킬 수 있는 클라우드 컴퓨팅!!

</aside>

### 🚀 **비즈니스 민첩성**

- **개발 속도 향상**: 서버 준비 시간 몇 주 → 몇 분
- **실험 용이**: 아이디어를 빠르게 테스트하고 검증 가능
- **시장 진입 속도**: 제품을 더 빨리 출시 가능

### 💸 **비용 절감**

- **초기 투자 제로**: 서버, 네트워크 장비 구매 불필요
- **운영비 최적화**: 사용한 만큼만 지불
- **인력 절약**: 서버 관리 인력 불필요

### 🔧 **운영 효율성**

- **핵심 업무 집중**: 서버 관리 대신 서비스 개발에 집중
- **자동 관리**: 백업, 보안 업데이트 등 자동화
- **전문성 활용**: 클라우드 업체의 전문 지식 활용

### 🌍 **글로벌 서비스**

- **전 세계 배포**: 클릭 몇 번으로 해외 진출
- **빠른 속도**: 사용자와 가까운 서버에서 서비스 제공
