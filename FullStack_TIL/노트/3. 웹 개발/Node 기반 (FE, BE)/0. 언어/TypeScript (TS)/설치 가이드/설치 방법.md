npm init -y

-D = --save-dev 
=> 개발 당시에만 사용하는 라이브러리라는 뜻.
=> 배포 시에는 사용되지 않는 라이브러리
```bash
# 1. 프로젝트 폴더 만들기 
mkdir topic-typescript-basic 
cd topic-typescript-basic 
# 2. nodeJS 프로젝트 셋업하기 
npm init -y 
# 3. Typescript 설치하기(-D : --save-dev) 
npm i -g typescript 
npm i -D typescript 
# 4. tsconfig.json 파일 생성 
npx tsc --init 
# 5. package.json 의 scripts 에 명령어 추가 
"scripts": { 
	"build": "tsc", 
	"start": "node main.js" 
}
```
