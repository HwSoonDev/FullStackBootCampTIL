#### 함수에 타입입히기

함수 타입 정의 후 변수에 함수 타입 지정 시 어떤 함수가 할당될 수 있고, 어떤 함수가 할당될 수 없는 지 기준을 아는 것이 중요합니다!
#### 1. 함수 타입 표현식

함수 타입 표현식은 **화살표 함수 문법과 유사한 형태로 함수의 타입을 정의**하는 방법입니다.

```ts
// 기본 형태: (매개변수: 타입) => 반환타입
type GreetFunction = (name: string) => string;

const greet: GreetFunction = (name) => {
  return `안녕하세요, ${name}님!`;
};

// 여러 매개변수가 있는 경우
type AddFunction = (a: number, b: number) => number;
const add: AddFunction = (a, b) => a + b;

// 매개변수가 없는 경우
type GetRandomNumber = () => number;
const getRandom: GetRandomNumber = () => Math.random();
```

#### 2. 호출 시그니처 (Call Signature)

호출 시그니처는 **객체 타입 내에서 함수의 타입을 정의**하는 방법입니다.

자바스크립트에서 함수는 일급 객체! 즉, 객체이므로 별도의 속성을 추가하거나 가질 수 있습니다.

```ts
interface DescribableFunction {
	// 함수 객체에 붙는 속성(함수 객체에 description이 있어야 한다는 뜻
	// ts에서 함수도 객체니까 속성을 붙일 수 있죠)
	description: string;
	
	// 함수 자체의 시그니처(숫자를 받아, boolean을 반환하는 함수라는 의미)
  (someArg: number): boolean;
};

function doSomething(fn: DescribableFunction) {
  console.log(fn.description + " 결과: " + fn(6));
}

// 사용 예시
function myFunc(n: number): boolean {
  return n > 5;
}
myFunc.description = "숫자가 5보다 큰지 확인";

doSomething(myFunc);
```

#### 기본 원칙

#### 1. 매개변수: "적을수록 좋다"

할당받는 함수가 더 적은 매개변수를 받아야 함

```ts
type Handler1 = (a: string) => void;
type Handler2 = (a: string, b: number) => void;

let h1: Handler1 = (a) => {};
let h2: Handler2 = (a, b) => {};

h1 = h2;// ❌ 에러! h2는 매개변수가 더 많음
h2 = h1;// ✅ 성공! h1이 매개변수가 더 적음// 실제 예시

// forEach는 (value, index, array) => void 타입으로 매개변수3개!
// 매개변수가 적은 함수도 더 많은 매개변수를 받는 자리에 들어갈 수 있다
const numbers = [1, 2, 3];
numbers.forEach((value) => console.log(value));// ✅
numbers.forEach((value, index) => console.log(value, index));// ✅
```

#### 2. 반환 타입: "더 구체적이어야 한다"

할당하는 함수의 반환 타입이 더 구체적(서브타입) 이어야 함

```ts
type Animal = { name: string };
type Dog = { name: string; breed: string };

type GetAnimal = () => Animal;
type GetDog = () => Dog;

let getAnimal: GetAnimal = () => ({ name: "동물" });
let getDog: GetDog = () => ({ name: "멍멍이", breed: "골든리트리버" });

getAnimal = getDog;// ✅ 성공! Dog는 Animal보다 구체적
getDog = getAnimal;// ❌ 에러! Animal은 Dog보다 추상적// 실제 예시
function createDog(): Dog {
  return { name: "멍멍이", breed: "골든리트리버" };
}

const animalFactory: GetAnimal = createDog;// ✅ 가능
```

#### 함수 오버로딩 (Function Overloading)

함수 오버로딩은 같은 이름의 함수를 여러 버전으로 정의하는 것입니다. 하나의 함수가 여러 개의 다른 함수 타입(매개변수 타입/개수)을 가질 수 있게 하는 기능

```ts
// 오버로드 시그니처들
function process(input: string): string;
function process(input: number): number;

// 구현 함수 (별도로 또 작성해야 함)
function process(input: string | number): string | number {
  if (typeof input === "string") return input.toUpperCase();
  return input * 2;
}

const result1 = process("hello");      // 타입: string
console.log(result1);                  // "HELLO"

const result2 = process(42);           // 타입: number
console.log(result2);                  // 84
```
