

Node.js는 브라우저 밖에서 자바스크립트를 실행할 수 있게 해주는 런타임 환경이에요.

- 크롬의 V8 자바스크립트 엔진을 사용하고
- 파일 시스템, 네트워크, 프로세스 등 OS 레벨 기능을 자바스크립트로 다룰 수 있게 해줘요.
- 특징:
- 이벤트 기반, 논블로킹 I/O: 많은 요청을 동시에 처리하는 백엔드 API 서버에 잘 맞음
- 싱글 스레드지만, 내부적으로 I/O는 스레드 풀을 활용해서 처리

그래서 지금 만들고 있는 Express 기반 백엔드도 사실은 Node.js 위에서 돌아가는 자바스크립트/타입스크립트 앱이라고 보면 됩니다.

---
javascript는 브라우저에서 돌아가기 위해 만들어진 언어입니다.
하지만, 브라우저 이외에 환경에서도 정말 유용한 언어였고 다른 분야에 사용하고 싶어졌습니다.
그래서 탄생한 것이 Node.js입니다.
