
빌드하기

쉽게 말해, 빌드된 소스들은 웹 서버에 배포해서 브라우저를 통해 접속할 수 있게 된다

로컬에 띄우면 로컬에서만, 웹에 띄우면 웹에서 접속이 가능하다
