Mogoose 스키마 정의에 validator를 넣어서 유효성 검사를 할 수 있다.
```js
name:{
        type: String,
        reqired: true,
        maxLength: 20,
        validate:{
          validator: function (title) {
            return title.split(' ').length > 1;
          },
          message: 'Must contain at least 2 words.'
        }
    },
```
