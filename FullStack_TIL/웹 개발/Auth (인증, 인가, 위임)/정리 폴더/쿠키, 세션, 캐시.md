
## 1) 쿠키 - 민감하지 않은 (브라우저)
사용자의 **브라우저**가 들고 있는 정보
도메인 별로 관리
사용자가 들고 있어도 되는 **민감하지 않은** 사용자 정보를 저장.

예시) 로그인 자동 저장, 세션 토큰, 방문기록

## 2) 세션 - 민감한 (서버)

**서버**에서 저장하는 사용자 정보 기록
사용자가 임의로 수정할 수 없어야 하는 **민감한 정보**를 저장.

**인증, 인가**에 주로 쓰인다.

로그인 세션을 발급해주면 쿠키에 저장해서 페이지 이동 시에도 기능 이용 유지

## 3) 캐시 - 재사용

재방문 등 **다시 사용될 확률이 높은 데이터**를 브라우저에 임시로 저장하는 것.
단순히 **로딩 속도를 보완**하고자 사용.
