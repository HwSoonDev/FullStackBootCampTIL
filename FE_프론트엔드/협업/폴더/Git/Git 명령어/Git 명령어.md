깃(Git) 로컬 작업에 사용하는 명령어를 모아두었다.

**명령어 색인 (help)**
1) git help add
2) man git-add
=> 둘 다 help의 기능을 한다.

**명령어**
1. [[git init]]                                               (깃 작업 영역 활성화)
2. [[git config]]                                          (작업자 정보 기록)
3. [[git add]]                                              (staging에 올리기)
4. [[git commit]]                                        (repo에 올리기)
5. [[git reset]]                                            (해당 커밋 형태로 되돌리기)
