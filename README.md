## 옵시디언으로 기록하는 이유

옵시디언 그래프/폴더 예시

옵시디언 폴더를 그대로 올려 두는 방식으로 TIL을 관리합니다. GitHub에서 링크만으로는 읽기 어렵지만, 옵시디언을 쓰는 이유는 다음과 같습니다.

1. 개념 연결을 그래프(마인드맵)로 볼 수 있다.
2. 작성·수정 시각이 자동으로 남는다.

**로컬에서 보기**: 저장소를 클론한 뒤 `FullStack_TIL` 폴더를 옵시디언 볼트로 열면 됩니다.
